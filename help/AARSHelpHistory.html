AARS Release History

BIT-ARM 0.1 was released in April 2020. Function followed are impleamented:

  • Frame has been modified to support ARM7
  • Basic instructions of ARM7 is mainly supported.
  • A few pseudo-instructions are added and tested
  • Sereval tools have been modified and tested
  • Directives have been modified and tested(multiple file not supported)

BIT-ARM 0.1 has limits follow:

  • only support User mode
  • unpredictable exceptions when using syscall or other function involve coprocessor1 in MARS(float)
  • pesudo_inst is far from enough for using (working on it now)
  • Macro is tested usable under a serial limits, not stable in widely use
  • use of register does not follow the general rules, syscall, for example

BIT-AARS 0.2 was released in April 2020. We changed the name of our software from 'ARM' to 'AARS', which means Advanced RISC Machine Assembly and Runtime Simulator. New functions followed are impleamented:

  • New GUI upgrade!
  • More instructions of ARM7 is added to our set and can be supported.
  • Add two example for showing how to use our BIT-AARS.
